# useless-object-inheritance

Class inherits from object, can be safely removed from bases in python3.

```grit
engine marzano(0.1)
language python

`class $name(object): $body` => `class $name: \n    $body`
```

## Example

```python
class Foo1:
    """Foo1 class"""

    class Foo2(object):
        """Foo2 class"""

        @staticmethod
        def hello():
            """say hello"""
            print("Hello!")
```

## Result

```python
class Foo1:
    """Foo1 class"""

    class Foo2: 
        """Foo2 class"""
    
        @staticmethod
        def hello():
            """say hello"""
            print("Hello!")
```

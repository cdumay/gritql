# use-dict-literal

Consider using '{}' instead of a call to 'dict'.

```grit
engine marzano(0.1)
language python

or {
  `dict()` => `{}`,
  `dict($data)` where {
    $values = [],
    $data <: some bubble($values) `$k=$v` where {
      $values += `"$k": $v`
    },
    $data <: maybe contains `**$kwargs` where {
      $values += `**$kwargs`
    },
    $new_data = join(list=$values, separator=", ")
  } => `{$new_data}`
}
```

## Example

```python
d1 = dict()
d2 = dict(a=5, b="foo")
d3 = dict(c=8, **d2)
```

## Result

```python
d1 = {}
d2 = {"a": 5, "b": "foo"}
d3 = {"c": 8, **d2}
```
